Standalone static helper at [/apps/crossword-helper/](/apps/crossword-helper/).

Drag the **length** slider to set how many letters the answer has (2–15), then
type the letters you already know into the matching boxes. Leave the unknown
positions blank. The full ~380k-word merged dictionary is filtered in real time
as you type, with the letters you pinned highlighted in green.

Heads-up: the dictionary only contains **standalone English words**. Crossword
answers often include proper nouns, names, places, brand names, titles,
phrases, hyphenated compounds, and modern slang — none of which appear here.
If a clue points to one of those, this tool can't help.

Same dataset and same in-browser, no-tracking approach as
[Word Explorer](/apps/dictionary/) and [Wordle Helper](/apps/wordle-helper/).
