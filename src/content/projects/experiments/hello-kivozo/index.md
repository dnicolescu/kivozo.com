# Hello, kivozo 👋

This is the first entry. If you're reading this, the content collection,
the category routing, and the homepage auto-index all work.

## How to add your own

1. Create a folder: `src/content/projects/<category>/<slug>/`
2. Add `index.md` with frontmatter (see this file for an example).
3. Either write the project inline in markdown, **or** set `href: /something/`
   (or a full URL) to point to a standalone app/page.

That's it — the homepage will pick it up on the next build.
